### Java Blackjack Game

A console-based Blackjack game developed in Java using object-oriented programming principles. The game simulates a real Blackjack round between a player and a dealer, including card dealing, hand evaluation, and win/loss logic based on standard casino rules.

#### Features

Fully implemented 52-card deck with shuffling and dealing

Player actions: hit or stand via console input
Dealer logic that follows Blackjack rules (hits until 17)
Automatic handling of Ace values (1 or 11)
Real-time detection of busts, wins, losses, and ties (push)

#### Technologies Used
Java
Object-Oriented Programming (OOP)
Collections Framework

#### Concepts Applied
Class design and encapsulation
Game logic and control flow
List manipulation and randomization
Input handling with Scanner
